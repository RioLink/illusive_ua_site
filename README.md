# Illusive UA static site

Статичний багатосторінковий сайт для digital/design студії.

## Сторінки

- `index.html` — головна сторінка
- `services.html` — послуги
- `portfolio.html` — портфоліо з фільтрами та галереєю
- `about.html` — підхід студії
- `contact.html` — контакти, форма та FAQ
- `privacy-policy.html` — політика конфіденційності
- `terms-and-conditions.html` — умови використання

## Ассети

- `assets/images/service/` — зображення для послуг
- `assets/images/portfolio/` — роботи портфоліо
- `assets/css/style.css` — стилі
- `assets/js/main.js` — меню, анімації, FAQ, cookie та фільтри портфоліо
